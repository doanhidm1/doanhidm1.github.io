# PSD.getBlend

機能/意味
:   合成結果の取得。取得領域は画像全体サイズ内におさまってる必要があります

タイプ
:   [PSDクラス](class_PSD)のメソッド

構文
:   getBlend(layer, left, top, width, height)

引数
:   |  |  |
    | --- | --- |
    | layer | 格納先レイヤ(width,heightサイズに調整される) |
    | left | 合成結果取得領域の左上座標 |
    | top | 合成結果取得領域の左上座標 |
    | width | 取得サイズ横幅 |
    | height | 取得サイズ縦幅 |

戻り値
:   取得に成功したら true
