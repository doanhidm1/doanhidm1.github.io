# WIN32Dialog.setScrollInfo

機能/意味
:   ScrollInfo の設定

タイプ
:   [WIN32Dialogクラス](class_WIN32Dialog)のメソッド

構文
:   setScrollInfo(id, pos, min, max, page)

引数
:   |  |  |
    | --- | --- |
    | id | スクロールバーのダイアログアイテムID |
    | pos | スライダーの位置 |
    | min | 取りうる最小の値 |
    | max | 取りうる最大の値 |
    | page | ページの値 |

戻り値
:   なし (void)
