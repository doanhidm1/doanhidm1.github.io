# WIN32Dialog.makeTemplate

機能/意味
:   ダイアログテンプレートを生成する

タイプ
:   [WIN32Dialogクラス](class_WIN32Dialog)のメソッド

構文
:   makeTemplate(header, items)

引数
:   |  |  |
    | --- | --- |
    | header | ダイアログテンプレートヘッダ （WIN32Dialog.Headerクラスのインスタンスであること） |
    | items\* | ダイアログテンプレートアイテム（WIN32Dialog.Items クラスのインスタンスであること） |

戻り値
:   なし (void)

説明
