# Scripts.equalStructNumericLoose

機能/意味
:   オブジェクト同士を比較する。辞書/配列の場合は中身の要素が再帰的に比較されます 数値がゆるく比較されます(0.0 と 0 を等しいとして扱います)

タイプ
:   [Scriptsクラス](class_Scripts)のメソッド

構文
:   equalStructNumericLoose(obj1, obj2)

引数
:   |  |  |
    | --- | --- |
    | obj1 | オブジェクトその1 |
    | obj2 | オブジェクトその2 |

戻り値
:   等しければ true

説明
