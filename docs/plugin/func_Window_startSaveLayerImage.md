# Window.startSaveLayerImage

機能/意味
:   TLG5/PNG 形式での画像の保存の開始

タイプ
:   [Windowクラス](class_Window)のメソッド

構文
:   startSaveLayerImage(layer, filename, tags)

引数
:   |  |  |
    | --- | --- |
    | layer | 保存対象レイヤ |
    | filename | ファイル名（拡張子が.pngの時のみPNG形式保存，それ以外はTLG5） |
    | tags | タグ情報 |

戻り値
:   ハンドラ

説明
