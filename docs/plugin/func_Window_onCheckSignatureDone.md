# Window.onCheckSignatureDone

機能/意味
:   イベント:署名チェック終了

タイプ
:   [Windowクラス](class_Window)のメソッド

構文
:   onCheckSignatureDone(handler, info, result, errormsg)

引数
:   |  |  |
    | --- | --- |
    | handler | 識別ハンドラ |
    | info | ユーザ情報 |
    | result | 結果 -2:エラー -1:キャンセル 0:チェック失敗 1:チェック成功 |
    | errormsg | 結果がエラーの場合のエラーメッセージ |

戻り値
:   なし (void)

説明
