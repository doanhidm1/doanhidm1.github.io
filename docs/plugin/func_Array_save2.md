# Array.save2

機能/意味
:   データを save に準じる形式で保存する

タイプ
:   [Arrayクラス](class_Array)のメソッド

構文
:   save2(filename, utf8, newline)

引数
:   |  |  |
    | --- | --- |
    | filename | 格納先ファイル |
    | utf8 | 出力エンコーディング指定。true なら UTF-8、falseなら現在のコードページ |
    | newline | 改行コード 0:CRLF 1:LF |

戻り値
:   なし (void)

説明
