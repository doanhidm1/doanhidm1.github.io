# ActiveX

ActiveX を操作できるオブジェクトを生成します。
ActiveX が IDispatch で持つメソッドを直接呼び出すことができます。

# メンバ

コンストラクタ
:   [ActiveX](func_ActiveX_ActiveX)

メソッド
:   [onCreate](func_ActiveX_onCreate) (ウインドウ生成完了後に呼び出されるイベント )
    [onDestroy](func_ActiveX_onDestroy) (ウインドウ破棄前に呼び出されるイベント )
    [setExternalUI](func_ActiveX_setExternalUI) (拡張UI指定 )
    [setPos](func_ActiveX_setPos) (座標指定 )
    [setSize](func_ActiveX_setSize) (サイズ指定 )

プロパティ
:   [height](prop_ActiveX_height) (縦幅 )
    [isValidWindow](prop_ActiveX_isValidWindow) (ウインドウが有効かどうか(読み込み専用) )
    [left](prop_ActiveX_left) (左端座標 )
    [top](prop_ActiveX_top) (上端座標 )
    [visible](prop_ActiveX_visible) (表示状態 )
    [width](prop_ActiveX_width) (横幅 )
