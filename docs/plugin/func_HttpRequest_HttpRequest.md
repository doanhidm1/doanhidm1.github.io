# HttpRequest.HttpRequest

機能/意味
:   コンストラクタ

タイプ
:   [HttpRequestクラス](class_HttpRequest)のメソッド

構文
:   HttpRequest(window, cert, agentName)

引数
:   |  |  |
    | --- | --- |
    | window | スレッド同期処理用親ウインドウ |
    | cert | HTTP通信時に証明書チェックを行うかどうか。falseにすると強制許可になります。 |
    | agentName | HTTP通信時のエージェント名 |

戻り値
:   なし (void)
