# Window.Window

機能/意味
:   コンストラクタ

タイプ
:   [Windowクラス](class_Window)のメソッド

構文
:   Window(window, left, top, width, height)

引数
:   |  |  |
    | --- | --- |
    | window | 親になるウインドウ |
    | left | 配置位置X |
    | top | 配置位置Y |
    | width | 配置サイズ |
    | height | 配置サイズ  生成した時点では Irrlichtのデバイスは存在しません。 生成後の最初のイベント処理前に初期化されて、 ドライバ、シーンマネージャ、GUIが扱えるようになります。 処理は onAttach() イベントを待ってから行う必要があります。 |

戻り値
:   なし (void)
