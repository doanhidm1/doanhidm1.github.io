# Path.drawPie

機能/意味
:   円錐の描画

タイプ
:   [Pathクラス](class_Path)のメソッド

構文
:   drawPie(x, y, width, height, startAngle, sweepAngle)

引数
:   |  |  |
    | --- | --- |
    | x | 左上座標 |
    | y | 左上座標 |
    | width | 横幅 |
    | height | 縦幅 |
    | startAngle | 時計方向円弧開始位置 |
    | sweepAngle | 描画角度 |

戻り値
:   なし (void)
