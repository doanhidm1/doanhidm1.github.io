# Window.commandSendSignal

機能/意味
:   Ctrl+Cイベントの送信(experimental:すべての環境で動作するとは限りません)

タイプ
:   [Windowクラス](class_Window)のメソッド

構文
:   commandSendSignal(process, isBreak)

引数
:   |  |  |
    | --- | --- |
    | process | プロセスハンドル（commandExecuteしたプロセスのみ） |
    | isBreak | Ctrl+CではなくCtrl+Breakを送る |

戻り値
:   成功したらtrue
