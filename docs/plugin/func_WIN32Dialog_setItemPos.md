# WIN32Dialog.setItemPos

機能/意味
:   アイテムの位置・大きさを変更・取得する

タイプ
:   [WIN32Dialogクラス](class_WIN32Dialog)のメソッド

構文
:   setItemPos(id, x, y)

引数
:   |  |  |
    | --- | --- |
    | id | ダイアログアイテムのID |
    | x | ダイアログアイテムのX座標 |
    | y | ダイアログアイテムのY座標 |
    | w | ダイアログアイテムの幅 |
    | h | ダイアログアイテムの高さ |

戻り値
:   なし (void)

説明
