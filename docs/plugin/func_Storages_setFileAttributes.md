# Storages.setFileAttributes

機能/意味
:   ファイルの属性を設定する

タイプ
:   [Storagesクラス](class_Storages)のメソッド

構文
:   setFileAttributes(filename, attr)

引数
:   |  |  |
    | --- | --- |
    | fliename | ファイル/ディレクトリ名 |
    | attr | 設定する属性(FILE\_ATTRIBUTE\_READONLY,FILE\_ATTRIBUTE\_HIDDEN,...) |

戻り値
:   なし (void)

説明
