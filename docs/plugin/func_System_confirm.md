# System.confirm

機能/意味
:   確認用メッセージ窓を表示します。ウインドウはモーダルで表示されます。

タイプ
:   [Systemクラス](class_System)のメソッド

構文
:   confirm(text, caption, window)

引数
:   |  |  |
    | --- | --- |
    | text | 表示するメッセージ |
    | caption | ウインドウのキャプション文字列 |
    | window | 指定があった場合はそのウインドウを親として表示します |

戻り値
:   YESがおされたら true

説明
