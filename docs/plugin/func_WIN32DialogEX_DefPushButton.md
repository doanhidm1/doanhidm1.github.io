# WIN32DialogEX.DefPushButton

機能/意味
:   各種コントロールを生成するための関数

タイプ
:   [WIN32DialogEXクラス](class_WIN32DialogEX)のメソッド

構文
:   DefPushButton(text, id, x, y, w, h, style, ex)

引数
:   |  |  |
    | --- | --- |
    | text | テキスト |
    | id | コントロールID |
    | x | x座標（ダイアログ座標単位） |
    | y | y座標（ダイアログ座標単位） |
    | w | 横幅 （ダイアログ座標単位） |
    | h | 高さ （ダイアログ座標単位） |
    | style | スタイル |
    | ex | 拡張スタイル |

戻り値
:   なし (void)
