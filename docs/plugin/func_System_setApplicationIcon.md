# System.setApplicationIcon

機能/意味
:   タスクバーやALT+TABなどで表示されるアプリケーションアイコンを変更する

タイプ
:   [Systemクラス](class_System)のメソッド

構文
:   setApplicationIcon(file)

引数
:   |  |  |
    | --- | --- |
    | file | アイコンファイルまたはvoid (voidか""で初期状態にリセット） |

戻り値
:   なし (void)

説明
:   ※ アーカイブ内のファイル指定は不可，exeやDLLを指定した場合は一番最初のアイコンが設定される
