# Layer.drawPathString

機能/意味
:   文字列のパスベースでの描画

タイプ
:   [Layerクラス](class_Layer)のメソッド

構文
:   drawPathString(font, app, x, y, text)

引数
:   |  |  |
    | --- | --- |
    | font | フォント GdiPlus.Font でフォントを指定します |
    | app | アピアランス GdiPlus.Appearance で描画スタイルを指定します |
    | x | 描画位置X |
    | y | 描画位置Y |
    | text | 描画テキスト |

戻り値
:   更新域情報(RectF)
