# WIN32DialogEX.loadResource

機能/意味
:   ダイアログリソースを読み込む（オーバーライド）

タイプ
:   [WIN32DialogEXクラス](class_WIN32DialogEX)のメソッド

構文
:   loadResource(dllfile, resource)

引数
:   |  |  |
    | --- | --- |
    | dllfile | dllファイル（自動的にStorages.getPlacedPath/getLocalName処理される） |
    | resource | ダイアログリソース名 |

戻り値
:   なし (void)

説明
