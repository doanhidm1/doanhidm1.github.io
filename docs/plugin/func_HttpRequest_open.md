# HttpRequest.open

機能/意味
:   指定したメソッドで指定URLにリクエストする

タイプ
:   [HttpRequestクラス](class_HttpRequest)のメソッド

構文
:   open(method, url, userName, password)

引数
:   |  |  |
    | --- | --- |
    | method | アクセスメソッド |
    | url | リクエスト先のURL |
    | userName | ユーザ名。指定すると認証ヘッダをつけます |
    | password | パスワード |

戻り値
:   なし (void)

説明
:   ※常に非同期での呼び出しになります
