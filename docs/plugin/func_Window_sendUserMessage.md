# Window.sendUserMessage

機能/意味
:   低レベルメッセージ送信を実行します。

タイプ
:   [Windowクラス](class_Window)のメソッド

構文
:   sendUserMessage(msg, wparam, lparam)

引数
:   |  |  |
    | --- | --- |
    | msg | メッセージ番号 |
    | wparam | WPARAM (整数扱いで処理されます) |
    | lparam | LPARAM (整数扱いで処理されます) |

戻り値
:   なし (void)

説明
:   全吉里吉里に指定のメッセージが送信されます
