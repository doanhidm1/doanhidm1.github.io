# Scripts.evalJSONStorage

機能/意味
:   JSON のファイルを解析して Array または Dictionary を返す

タイプ
:   [Scriptsクラス](class_Scripts)のメソッド

構文
:   evalJSONStorage(jsonFile, utf8)

引数
:   |  |  |
    | --- | --- |
    | jsonFile | JSON が格納されたテキストファイルのファイル名 |
    | utf8 | 入力エンコーディング指定。true なら UTF-8、falseなら現在のコードページ |

戻り値
:   Array または Dictionary
