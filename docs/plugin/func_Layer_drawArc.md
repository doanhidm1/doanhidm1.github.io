# Layer.drawArc

機能/意味
:   円弧の描画

タイプ
:   [Layerクラス](class_Layer)のメソッド

構文
:   drawArc(app, x, y, width, height, startAngle, sweepAngle)

引数
:   |  |  |
    | --- | --- |
    | app | アピアランス GdiPlus.Appearance で描画スタイルを指定します |
    | x | 左上座標 |
    | y | 左上座標 |
    | width | 横幅 |
    | height | 縦幅 |
    | startAngle | 時計方向円弧開始位置 |
    | sweepAngle | 描画角度 |

戻り値
:   更新領域情報の辞書 left, top, width, height
