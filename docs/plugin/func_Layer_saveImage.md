# Layer.saveImage

機能/意味
:   画像の保存

タイプ
:   [Layerクラス](class_Layer)のメソッド

構文
:   saveImage(filename, mime, param)

引数
:   |  |  |
    | --- | --- |
    | filename | ファイル名 |
    | mime | 保存する画像形式 image/jpeg, image/png, image/gif など |
    | param | エンコードパラメータ(辞書) compression, scanmethod, version, render, transform, quality(0～100), depth(1,4,8,24,32) |

戻り値
:   なし (void)
