# Storages.resetFileAttributes

機能/意味
:   ファイルの属性を解除する

タイプ
:   [Storagesクラス](class_Storages)のメソッド

構文
:   resetFileAttributes(filename, attr)

引数
:   |  |  |
    | --- | --- |
    | fliename | ファイル/ディレクトリ名 |
    | attr | 解除する属性(FILE\_ATTRIBUTE\_READONLY,FILE\_ATTRIBUTE\_HIDDEN,...) |

戻り値
:   なし (void)

説明
