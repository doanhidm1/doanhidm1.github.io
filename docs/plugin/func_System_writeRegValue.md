# System.writeRegValue

機能/意味
:   レジストリにデータを書き込みます

タイプ
:   [Systemクラス](class_System)のメソッド

構文
:   writeRegValue(key, value)

引数
:   |  |  |
    | --- | --- |
    | key | 書き込み先のキー(readRegValue と同様に HKEY\_LOCAL\_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\hoeg\installdir と指定する) |
    | value | 書き込む値(文字列又は整数値) |

戻り値
:   なし (void)

説明
