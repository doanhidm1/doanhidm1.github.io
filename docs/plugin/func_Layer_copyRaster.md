# Layer.copyRaster

機能/意味
:   ラスターコピー処理

タイプ
:   [Layerクラス](class_Layer)のメソッド

構文
:   copyRaster(src, maxh, lines, cycle, time)

引数
:   |  |  |
    | --- | --- |
    | src | 描画元レイヤ。自分と同じサイズでないと処理が行われません。 |
    | maxh | 最大振幅(pixel) |
    | line | １周期あたりライン数 |
    | cycle | 周期指定(msec) |
    | time | 現在時刻 |

戻り値
:   なし (void)

説明
