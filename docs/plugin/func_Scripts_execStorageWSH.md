# Scripts.execStorageWSH

機能/意味
:   WSH スクリプトのファイルからの実行

タイプ
:   [Scriptsクラス](class_Scripts)のメソッド

構文
:   execStorageWSH(storage, extOrProgIdOrCLSID)

引数
:   |  |  |
    | --- | --- |
    | storage | スクリプトが格納されたファイル |
    | extOrProgIdOrCLSID | 言語を指定します。拡張子またはProgIdまたはCLSID 省略された場合は第一引数の拡張子から決定されます 指定されたストレージを読み込み、その内容を、指定された言語のスクリプトとして実行します。 |

戻り値
:   なし (void)

説明
