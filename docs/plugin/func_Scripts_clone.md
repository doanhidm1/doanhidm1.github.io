# Scripts.clone

機能/意味
:   値のクローンを返す。

タイプ
:   [Scriptsクラス](class_Scripts)のメソッド

構文
:   clone(obj)

引数
:   |  |  |
    | --- | --- |
    | obj | クローン元 void/数値/文字列/octet は元の値をそのまま返します。 辞書・配列オブジェクトは、deep copy 処理したものを返します。入れ子は再帰処理されます。 一般オブジェクトは "clone" メソッドが存在すればその帰り値を、なければ元の値をそのまま返します |

戻り値
:   なし (void)
