# WIN32Dialog.setItemBitmap

機能/意味
:   アイテムにBitmapを設定する

タイプ
:   [WIN32Dialogクラス](class_WIN32Dialog)のメソッド

構文
:   setItemBitmap(id, bmp)

引数
:   |  |  |
    | --- | --- |
    | id | ダイアログアイテムのID |
    | bmp | ビットマップに設定するWIN32Dialog.Bitmapのインスタンス WIN32DialogEXではLayerインスタンスが指定可能 |

戻り値
:   なし (void)

説明
