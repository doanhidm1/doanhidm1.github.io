# Scripts.compileSQ

機能/意味
:   Squirrel スクリプトのコンパイル処理

タイプ
:   [Scriptsクラス](class_Scripts)のメソッド

構文
:   compileSQ(text, store, endian)

引数
:   |  |  |
    | --- | --- |
    | text | スクリプトが格納された文字列 |
    | store | バイナリクロージャ格納先ファイル |
    | endian | エンディアン指定 0:デフォルト 1:little 2:big |

戻り値
:   エラー文字列または void

説明
