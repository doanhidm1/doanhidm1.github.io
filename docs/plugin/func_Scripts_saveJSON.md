# Scripts.saveJSON

機能/意味
:   データを JSON 形式で保存する

タイプ
:   [Scriptsクラス](class_Scripts)のメソッド

構文
:   saveJSON(jsonFile, obj, utf8, newline)

引数
:   |  |  |
    | --- | --- |
    | jsonFile | 格納先ファイル |
    | obj | 保存対象オブジェクト |
    | utf8 | 出力エンコーディング指定。true なら UTF-8、falseなら現在のコードページ |
    | newline | 改行コード 0:CRLF 1:LF |

戻り値
:   なし (void)

説明
