# Scripts.compileStorageSQ

機能/意味
:   Squirrel スクリプトのファイルからのコンパイル処理

タイプ
:   [Scriptsクラス](class_Scripts)のメソッド

構文
:   compileStorageSQ(filename, store, endian)

引数
:   |  |  |
    | --- | --- |
    | filename | コンパイル元ファイル |
    | store | バイナリクロージャ格納先ファイル |
    | endian | エンディアン指定 0:デフォルト 1:little 2:big |

戻り値
:   エラー文字列または void

説明
