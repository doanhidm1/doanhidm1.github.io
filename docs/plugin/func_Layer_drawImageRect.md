# Layer.drawImageRect

機能/意味
:   画像の矩形コピー

タイプ
:   [Layerクラス](class_Layer)のメソッド

構文
:   drawImageRect(dleft, dtop, src, sleft, stop, swidth, sheight)

引数
:   |  |  |
    | --- | --- |
    | dleft | コピー先左端 |
    | dtop | コピー先上端 |
    | src | コピー元画像(Image/Layer/ファイル名) |
    | sleft | 元矩形の左端 |
    | stop | 元矩形の上端 |
    | swidth | 元矩形の横幅 |
    | sheight | 元矩形の縦幅 |

戻り値
:   更新域情報(RectF)
