# Scripts.execWSH

機能/意味
:   WSH スクリプトの実行

タイプ
:   [Scriptsクラス](class_Scripts)のメソッド

構文
:   execWSH(script, extOrProgIdOrCLSID)

引数
:   |  |  |
    | --- | --- |
    | script | スクリプトが格納された文字列 |
    | extOrProgIdOrCLSID | 言語を指定します。拡張子またはProgIdまたはCLSID 拡張子: js, vbs, pl/pls, rb/rbs のいずれか ProgId: JScript, VBScript, PerlScript, RubyScript など CLSID: スクリプトエンジンのクラスIDを指定します。 script で指定された文字列を指定された言語のスクリプトとして実行します。 |

戻り値
:   なし (void)

説明
