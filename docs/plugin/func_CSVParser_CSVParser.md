# CSVParser.CSVParser

機能/意味
:   コンストラクタ

タイプ
:   [CSVParserクラス](class_CSVParser)のメソッド

構文
:   CSVParser(target, separator, newline)

引数
:   |  |  |
    | --- | --- |
    | target | イベントを転送するターゲット。未指定時は自分に送る |
    | separator | 区切り文字。デフォルトは "," |
    | newline | 中途で区切られた場合のデータでの改行文字。デフォルトは "\r " |

戻り値
:   なし (void)

説明
