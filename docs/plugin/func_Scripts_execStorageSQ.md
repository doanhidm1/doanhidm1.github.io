# Scripts.execStorageSQ

機能/意味
:   Squirrel スクリプトのファイルからの実行。

タイプ
:   [Scriptsクラス](class_Scripts)のメソッド

構文
:   execStorageSQ(filename, ...)

引数
:   |  |  |
    | --- | --- |
    | filename | スクリプトが格納されたファイル |
    | ... | 引数 |

戻り値
:   スクリプトの実行結果

説明
:   このメソッドによる実行中では Object/Thread 拡張による wait は利用できません
