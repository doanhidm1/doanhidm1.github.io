# PSD.getLayerInfo

機能/意味
:   レイヤの情報の取得

タイプ
:   [PSDクラス](class_PSD)のメソッド

構文
:   getLayerInfo(no)

引数
:   |  |  |
    | --- | --- |
    | no | レイヤ番号 |

戻り値
:   レイヤ情報がはいった辞書\n辞書の内容\nlayer\_type レイヤ種別\ntop 上座標\nleft 左座標\nbottom 底座標\nright 右座標\nwidth 横幅\nheight 縦幅\nblend\_mode 合成モード\nopacity 不透明度\nvisible 表示状態\nname レイヤ名\ntype 合成モード（吉里吉里の対応モード)
