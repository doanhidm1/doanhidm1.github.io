# Layer.drawString

機能/意味
:   文字列の描画。描画のクオリティは textRenderingHint に従います。

タイプ
:   [Layerクラス](class_Layer)のメソッド

構文
:   drawString(font, app, x, y, text)

引数
:   |  |  |
    | --- | --- |
    | font | フォント GdiPlus.Font でフォントを指定します |
    | app | アピアランス GdiPlus.Appearance で描画スタイルを指定します。※ブラシしか参照されません |
    | x | 描画位置X |
    | y | 描画位置Y |
    | text | 描画テキスト |

戻り値
:   更新域情報(RectF)
