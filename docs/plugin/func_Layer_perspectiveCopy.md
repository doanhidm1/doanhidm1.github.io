# Layer.perspectiveCopy

機能/意味
:   透視変形コピー

タイプ
:   [Layerクラス](class_Layer)のメソッド

構文
:   perspectiveCopy(src, left, top, width, height, x1, y1, x2, y2, x3, y3, x4, y4)

引数
:   |  |  |
    | --- | --- |
    | src | コピー元レイヤ |
    | left | コピー元 left |
    | top | コピー元 top |
    | width | コピー元 width |
    | height | コピー元 height |
    | x1 | 左上コピー先 |
    | y1 | 左上コピー先 |
    | x2 | 右上コピー先 |
    | y2 | 右上コピー先 |
    | x3 | 左下コピー先 |
    | y3 | 左下コピー先 |
    | x4 | 右下コピー先 |
    | y4 | 右下コピー先 |

戻り値
:   なし (void)
