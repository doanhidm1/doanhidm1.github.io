# Window.BasicDrawDevice

Window.BasicDrawDevice クラスは、このインスタンスを [Window.drawDevice](f_Window_drawDevice) に登録して使用するための DrawDevice で、標準的な描画機能を提供します。

# メンバ

コンストラクタ
:   [BasicDrawDevice](f_Window.BasicDrawDevice_BasicDrawDevice)

メソッド
:   [recreate](f_Window.BasicDrawDevice_recreate) ( 内部デバイス再生成 )

プロパティ
:   [interface](f_Window.BasicDrawDevice_interface) ( インターフェースオブジェクトを取得 )

イベント
:   [onDisplayRotate](f_Window.BasicDrawDevice_onDisplayRotate) ( 画面が回転されたとき(1.1.0以降) )
