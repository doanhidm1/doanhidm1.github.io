# VideoOverlay.VideoOverlay

:   VideoOverlay オブジェクトの構築

タイプ
:   [VideoOverlayクラス](f_VideoOverlay)のコンストラクタ

引数
:   |  |  |
    | --- | --- |

戻り値

説明
