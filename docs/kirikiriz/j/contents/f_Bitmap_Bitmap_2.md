# Bitmap.Bitmap

機能/意味
:   Bitmap オブジェクトの構築

タイプ
:   [Bitmapクラス](f_Bitmap)のコンストラクタ

構文
:   Bitmap(filename, colorkey=clNone)

引数
:   |  |  |
    | --- | --- |
    | filename | 画像のファイル名を指定します。 |
    | colorkey | カラーキーを指定します。  値は [Layer.loadImages](f_Layer_loadImages) と同じです。 |

戻り値
:   なし (void)

説明
:   Bitmap クラスのオブジェクトをファイルから構築します。
