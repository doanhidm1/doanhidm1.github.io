# Window.close

機能/意味
:   ウィンドウを閉じる

タイプ
:   [Windowクラス](f_Window)のメソッド

構文
:   close()

引数
:   なし

戻り値
:   なし (void)

説明
:   [Window.showModal](f_Window_showModal) メソッドで表示されたウィンドウを閉じます。ウィンドウを閉じる前に [Window.onCloseQuery](f_Window_onCloseQuery) イベントが発生し、ウィンドウを閉じることができるかどうかを確認することができます。

参照

    [Window.onCloseQuery](f_Window_onCloseQuery)
