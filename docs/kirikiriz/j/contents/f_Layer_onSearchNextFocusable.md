# Layer.onSearchNextFocusable

機能/意味
:   フォーカス可能なレイヤの後方検索

タイプ
:   [Layerクラス](f_Layer)のイベント

構文
:   onSearchNextFocusable(layer)

引数
:   |  |  |
    | --- | --- |
    | layer | フォーカスを受け取るレイヤオブジェクトです。  　親クラスの同名のメソッドを呼ぶときに引数として指定します。  　null を渡すと、フォーカス可能なレイヤは無いとして処理されます。 |

説明
:   フォーカス可能なレイヤの後方検索を行うときに発生するイベントです。
    　フォーカスを受け取るレイヤを指定するときは、スーパークラスの同名のイベント
    の引数にそのレイヤを指定してください。
