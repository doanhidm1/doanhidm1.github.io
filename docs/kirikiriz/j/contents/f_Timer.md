# Timer

Timer クラスは、一定時間ごとにイベントを発生するためのクラスです。

# メンバ

コンストラクタ
:   [Timer](f_Timer_Timer)

メソッド
:   なし

プロパティ
:   [capacity](f_Timer_capacity) ( イベントの最大発生許容量 )
    [enabled](f_Timer_enabled) ( イベントを発生させるかどうか )
    [interval](f_Timer_interval) ( イベントの発生周期 )
    [mode](f_Timer_mode) ( モード )

イベント
:   [onTimer](f_Timer_onTimer) ( タイマー周期が来た )
