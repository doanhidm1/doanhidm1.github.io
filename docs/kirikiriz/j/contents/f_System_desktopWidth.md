# System.desktopWidth

機能/意味
:   デスクトップ幅

タイプ
:   [Systemクラス](f_System)のプロパティ (読み出し専用)

説明
:   デスクトップ ( ウィンドウを表示可能な領域 ) の幅をピクセル単位で表します。
    　値はメインウィンドウのあるディスプレイを対象としたものです。
    　メインウィンドウがない場合はプライマリーディスプレイが対象となります。
