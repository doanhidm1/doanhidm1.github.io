# Window.hideMouseCursor

機能/意味
:   マウスカーソルを一時的に隠す

タイプ
:   [Windowクラス](f_Window)のメソッド

構文
:   hideMouseCursor()

引数
:   なし

戻り値
:   なし (void)

説明
:   マウスカーソルを一時的に隠します。マウスを少しでも動かすと
    マウスカーソルは再び表示されるようになります。
    　このメソッドは、[Window.mouseCursorState](f_Window_mouseCursorState)をmcsTempHiddenに設定するのと同じ効果を持ちます。
