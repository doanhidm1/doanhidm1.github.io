# WaveSoundBuffer.open

機能/意味
:   メディアを開く

タイプ
:   [WaveSoundBufferクラス](f_WaveSoundBuffer)のメソッド

構文
:   open(storage)

引数
:   |  |  |
    | --- | --- |
    | storage | 再生したいストレージを指定します。 |

戻り値
:   なし (void)

説明
:   指定されたメディアを開きます。このメソッドは再生を開始しません。
    　指定されたストレージ名に .sli を付加したファイル名があれば、サウンドループ情報として読み込みます。
