# Rect.clip

機能/意味
:   矩形のクリッピング

タイプ
:   [Rectクラス](f_Rect)のメソッド

構文
:   clip(r)

引数
:   |  |  |
    | --- | --- |
    | r | クリッピングする矩形オブジェクトを指定します。 |

戻り値
:   クリッピング実施有無

説明
:   矩形のクリッピングを指定の矩形で行います。
    　矩形が空にならない場合のみクリッピングは行われます。
    　空になってしまう場合は false が返り、クリッピングは行われません。
