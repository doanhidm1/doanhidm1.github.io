# Layer.absoluteOrderMode

機能/意味
:   絶対位置モードかどうか

タイプ
:   [Layerクラス](f_Layer)のプロパティ (読み書き可能)

説明
:   直属の子レイヤの重ね合わせ順モードを表します。値を設定することもできます。
    　偽を指定すると相対位置指定となり、[Layer.order](f_Layer_order) プロパティが
    その順位を表すようになります。
    　真を指定すると絶対位置指定となり、[Layer.absolute](f_Layer_absolute) プロパティが
    その順位を表すようになります。
