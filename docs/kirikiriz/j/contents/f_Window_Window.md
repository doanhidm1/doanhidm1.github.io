# Window.Window

機能/意味
:   Window オブジェクトの構築

タイプ
:   [Windowクラス](f_Window)のコンストラクタ

構文
:   Window(parent)

引数
:   |  |  |
    | --- | --- |
    | parent | 親ウィンドウを指定します。  　親ウィンドウを指定すると、その子ウィンドウとして生成されます。  　指定しない場合は省略します。 |

戻り値
:   なし (void)

説明
:   Window クラスのオブジェクトを構築します。
    　ウィンドウは非表示の状態で作成され、位置やサイズは未定義 ( どこかにの位置に適当なサイズ ) です。
