# ImageFunction

ImageFunction クラスは、画像を操作するクラスです。
　このクラスからオブジェクトを作成することはできません。

# メンバ

:   なし

    [doGrayScale](f_ImageFunction_doGrayScale) ( グレースケール変換 )
    [drawGlyph](f_ImageFunction_drawGlyph) ( 文字描画 )
    [drawText](f_ImageFunction_drawText) ( 文字描画 )
    [fillRect](f_ImageFunction_fillRect) ( 矩形塗りつぶし )

    [flipUD](f_ImageFunction_flipUD) ( 上下反転 )

    [operateRect](f_ImageFunction_operateRect) ( 矩形演算合成 )
