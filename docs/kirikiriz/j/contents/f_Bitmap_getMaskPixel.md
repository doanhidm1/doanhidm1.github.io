# Bitmap.getMaskPixel

機能/意味
:   ピクセルのアルファ値の取得

タイプ
:   [Bitmapクラス](f_Bitmap)のメソッド

構文
:   getMaskPixel(x, y)

引数
:   |  |  |
    | --- | --- |
    | x | 取得する x 座標値をピクセル単位で指定します。 |
    | y | 取得する y 座標値をピクセル単位で指定します。 |

戻り値
:   ピクセルのアルファ値

説明
:   画像のピクセルのアルファ値を取得します。

参照
:   [Bitmap.setMaskPixel](f_Bitmap_setMaskPixel)
