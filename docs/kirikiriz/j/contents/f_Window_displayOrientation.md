# Window.displayOrientation

機能/意味
:   ディスプレイの向き(1.1.0以降)

タイプ
:   [Windowクラス](f_Window)のプロパティ (読み出し専用)

説明
:   ディスプレイの向きを表します。
    　oriUnknown (取得失敗/不明), oriPortrait(縦向き), oriLandscape(横向き)のいずれかの値です。
