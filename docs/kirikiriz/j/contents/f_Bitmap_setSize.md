# Bitmap.setSize

機能/意味
:   画像サイズの設定

タイプ
:   [Bitmapクラス](f_Bitmap)のメソッド

構文
:   setSize(width, height)

引数
:   |  |  |
    | --- | --- |
    | width | 画像の横幅をピクセル単位で指定します。  　この値は [Bitmap.width](f_Bitmap_width) プロパティでも取得や設定ができます。 |
    | height | 画像の縦幅をピクセル単位で指定します。  　この値は [Bitmap.height](f_Bitmap_height) プロパティでも取得や設定ができます。 |

戻り値
:   なし (void)

説明
:   画像サイズを指定します。
