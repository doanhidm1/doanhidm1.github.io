# Window.BasicDrawDevice.BasicDrawDevice

機能/意味
:   BasicDrawDevice オブジェクトの構築

タイプ
:   [Window.BasicDrawDeviceクラス](f_Window.BasicDrawDevice)のコンストラクタ

構文
:   BasicDrawDevice()

引数
:   なし

戻り値
:   なし (void)

説明
:   Window.BasicDrawDevice クラスのオブジェクトを構築します。
    　初期状態で Window.drawDevice にはこのクラスのインスタンスが登録されているので、新たに登録する必要はありません。
