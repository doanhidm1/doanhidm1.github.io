# Bitmap.setMaskPixel

機能/意味
:   ピクセルのアルファ値の取得

タイプ
:   [Bitmapクラス](f_Bitmap)のメソッド

構文
:   setMaskPixel(x, y, value)

引数
:   |  |  |
    | --- | --- |
    | x | 設定する x 座標値をピクセル単位で指定します。 |
    | y | 設定する y 座標値をピクセル単位で指定します。 |
    | value | 設定するピクセルのアルファ値を指定します。 |

戻り値
:   なし (void)

説明
:   画像のピクセルのアルファ値を設定します。

参照
:   [Bitmap.getMaskPixel](f_Bitmap_getMaskPixel)
