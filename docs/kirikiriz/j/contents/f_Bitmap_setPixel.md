# Bitmap.setPixel

機能/意味
:   ピクセル値の設定

タイプ
:   [Bitmapクラス](f_Bitmap)のメソッド

構文
:   setPixel(x, y, value)

引数
:   |  |  |
    | --- | --- |
    | x | 設定する x 座標値をピクセル単位で指定します。 |
    | y | 設定する y 座標値をピクセル単位で指定します。 |
    | value | 設定するピクセル値を指定します。 |

戻り値
:   なし (void)

説明
:   画像のピクセル値を設定します。

参照
:   [Bitmap.getPixel](f_Bitmap_getPixel)
