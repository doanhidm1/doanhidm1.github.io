# Storages

Storages クラスは 吉里吉里本体のストレージシステムに関するメソッドやプロパティを持ったクラスです。このクラスからオブジェクトを作成することはできません。

# メンバ

コンストラクタ
:   なし

メソッド
:   [addAutoPath](f_Storages_addAutoPath) ( 自動検索パスへの追加 )
    [chopStorageExt](f_Storages_chopStorageExt) ( ストレージ名の拡張子の切り落とし )
    [extractStorageExt](f_Storages_extractStorageExt) ( ストレージ名の拡張子の抽出 )
    [extractStorageName](f_Storages_extractStorageName) ( ストレージ名の抽出 )
    [extractStoragePath](f_Storages_extractStoragePath) ( ストレージ名のパスの抽出 )
    [getFullPath](f_Storages_getFullPath) ( 完全な統一ストレージ名の取得 )
    [getLocalName](f_Storages_getLocalName) ( ローカルファイル名の取得 )
    [getPlacedPath](f_Storages_getPlacedPath) ( ストレージの検索 )
    [isExistentStorage](f_Storages_isExistentStorage) ( ストレージの存在確認 )
    [removeAutoPath](f_Storages_removeAutoPath) ( 自動検索パスの削除 )
    [searchCD](f_Storages_searchCD) ( CD の検索 )
    [selectFile](f_Storages_selectFile) ( ファイル選択ダイアログボックスを表示 )

プロパティ
:   なし

イベント
:   なし
