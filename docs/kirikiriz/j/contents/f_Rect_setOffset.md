# Rect.setOffset

機能/意味
:   矩形位置の設定

タイプ
:   [Rectクラス](f_Rect)のメソッド

構文
:   setOffset(x, y)

引数
:   |  |  |
    | --- | --- |
    | x | 矩形の X 座標値を指定します。  　この値は [Rect.left](f_Rect_left) プロパティでも取得や設定ができます。 |
    | y | 矩形の Y 座標値を指定します。  　この値は [Rect.topt](f_Rect_topt) プロパティでも取得や設定ができます。 |

戻り値
:   なし (void)

説明
:   矩形位置の設定を行います。
