# Layer.focus

機能/意味
:   フォーカスの設定

タイプ
:   [Layerクラス](f_Layer)のメソッド

構文
:   focus(direction=true)

引数
:   |  |  |
    | --- | --- |
    | direction | [Layer.onBeforeFocus](f_Layer_onBeforeFocus) や [Layer.onFocus](f_Layer_onFocus) イベントの direction 引数に渡される値です。 |

戻り値
:   なし (void)

説明
:   レイヤにフォーカスを指定します。レイヤがフォーカスを受け取ると
    キーボード入力が可能になります。
