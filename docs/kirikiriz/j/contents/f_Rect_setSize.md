# Rect.setSize

機能/意味
:   矩形サイズの設定

タイプ
:   [Rectクラス](f_Rect)のメソッド

構文
:   setSize(width, height)

引数
:   |  |  |
    | --- | --- |
    | width | 矩形の横幅を指定します。  　この値は [Rect.width](f_Rect_width) プロパティでも取得や設定ができます。 |
    | height | 矩形の縦幅を指定します。  　この値は [Rect.height](f_Rect_height) プロパティでも取得や設定ができます。 |

戻り値
:   なし (void)

説明
:   矩形サイズを指定します。
