# Layer.copy9Patch

機能/意味
:   9 path を利用した画像コピー(1.3.0以降)

タイプ
:   [Layerクラス](f_Layer)のメソッド

構文
:   copy9Patch(src)

引数
:   |  |  |
    | --- | --- |
    | src | コピー元のレイヤオブジェクトを指定します。  　Bitmapクラスのオブジェクトも指定可能です。 |

戻り値
:   マージン情報

説明
:   9 path ( slice ) を利用した画像コピーを行います。
    　返されるマージン情報はRectクラスのオブジェクトです。
