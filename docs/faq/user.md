# ユーザー編 FAQ

初めに

ゲームを行うユーザーが遭遇する可能性のある問題についての FAQ です。

起動

Q.MFPlat.DLL や MF.dll が見つからないと表示される。
:   A.Windows Nエディションの場合マルチメディア関係の機能が含まれていないため起動できません。
    Media Feature Pack をインストールしてください。

サウンド

Q.ボイスが再生される時に最初にプツっと音がする
:   A.-wsrecreate=yes で直る場合があります。
    コマンドラインオプションで直指定するか、吉里吉里設定で「サウンド:DirectSoundセカンダリバッファ再生成：常に再再生成」に設定します。

マウス・入力

Q.ホイールを回してもメッセージが進まない、反応しない
:   A.-wheel で直る場合があります。
    dinput か message の両方を試してうまくいく方を選択します。
    コマンドラインオプションで直接指定するか、吉里吉里設定で「入力:マウスホイール検出方法:DirectInput」か「入力:マウスホイール検出方法:ウィンドウメッセージ」に設定します。

Q.何もしていないのに勝手にメッセージが進む。
:   A.ゲームパッドが接続されていて、そのボタンが押しっぱなしになっている場合、キーリピートでメッセージが進んでいることがあります。
    また、自動的に読み進むになっていると自動的に進みます。
