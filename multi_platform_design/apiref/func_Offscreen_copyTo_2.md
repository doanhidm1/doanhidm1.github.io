# Offscreen.copyTo

機能/意味
:   Bitmapにコピー

タイプ
:   [Offscreenクラス](class_Offscreen)のメソッド

構文
:   copyTo(destBitmap:Bitmap, dleft:int, dtop:int, srcRect:Rect)

引数
:   |  |  |
    | --- | --- |
    | destBitmap | 格納先ビットマップ |
    | dleft | コピー先左端 |
    | dtop | コピー先上端 |
    | srcRect | コピー元矩形 |

戻り値
:   なし (void)
