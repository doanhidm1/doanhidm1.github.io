# Array.loadStruct

機能/意味
:   ファイルから構造化されたデータの入力を行います

タイプ
:   [Arrayクラス](class_Array)のメソッド

構文
:   loadStruct(storage:string, mode:string=null) :Array

引数
:   |  |  |
    | --- | --- |
    | storage | 読み込むファイル名 |
    | mode | ファイルを読み込む際のモード文字列を指定します。   * "o" に続いてオフセットを10進で指定するとファイルのそのバイト位置からの書き込みになります。 |

戻り値
:   読み込みに成功した場合、この Array クラスのオブジェクト自身を返します。

説明
:   構造化されたデータをファイルから配列として読み込みます。
    バイナリ形式で書き出されたファイルのみ対応しています。
