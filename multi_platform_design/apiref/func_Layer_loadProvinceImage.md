# Layer.loadProvinceImage

機能/意味
:   領域画像の読み込み

タイプ
:   [Layerクラス](class_Layer)のメソッド

構文
:   loadProvinceImage(image)

引数
:   |  |  |
    | --- | --- |
    | image | 領域画像として読み込む画像ストレージを指定します。 |

戻り値
:   なし (void)

説明
:   レイヤの領域画像を読み込みます。
    それ以外の画像はそのままとなります。
    読み込もうとした画像がレイヤの画像サイズと異なる場合は例外が発生します。
