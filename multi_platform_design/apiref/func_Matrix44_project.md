# Matrix44.project

機能/意味
:   各種行列を用いた座標変換

タイプ
:   [Matrix44クラス](class_Matrix44)のメソッド

構文
:   project(model:Matrix44, proj:Matrix44, viewport:Rect, x:real, y:real, z:real)

引数
:   |  |  |
    | --- | --- |
    | model | モデル行列 |
    | proj | プロジェクション行列 |
    | viewport | ビューポート |
    | x | X座標値(in/out) |
    | y | Y座標値(in/out) |
    | z | Z座標値(in/out) |

戻り値
:   なし (void)

説明
:   クラスメソッドです。
