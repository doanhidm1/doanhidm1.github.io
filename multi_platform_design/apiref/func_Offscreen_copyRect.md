# Offscreen.copyRect

機能/意味
:   矩形コピー

タイプ
:   [Offscreenクラス](class_Offscreen)のメソッド

構文
:   copyRect(dleft:int, dtop:int, src:Bitmap, srcRect:Rect)

引数
:   |  |  |
    | --- | --- |
    | dleft | コピー先左端 |
    | dtop | コピー先上端 |
    | src | Bitmapクラスのオブジェクト |
    | srcRect | srcのコピー元矩形 |

戻り値
:   なし (void)
