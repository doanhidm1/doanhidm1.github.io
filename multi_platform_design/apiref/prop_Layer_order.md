# Layer.order

機能/意味
:   相対位置

タイプ
:   [Layerクラス](class_Layer)のプロパティ

説明
:   同じ親を持つ兄弟レイヤ間での順位を表します。
    値が小さいほど奥に表示されます。
    値を設定すると兄弟レイヤ間での順位を変えることができます。値を設定すると親レイヤの Layer.absoluteOrderMode プロパティが偽に設定されます。
