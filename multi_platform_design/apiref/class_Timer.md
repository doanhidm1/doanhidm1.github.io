# Timer

Timer クラスは、一定時間ごとにイベントを発生するためのクラスです。

# メンバ

コンストラクタ
:   [Timer](func_Timer_Timer) (Timer オブジェクトの構築 )

メソッド

プロパティ
:   [capacity](prop_Timer_capacity) (イベントの最大発生許容量 )
    [enabled](prop_Timer_enabled) (イベントを発生させるかどうか )
    [interval](prop_Timer_interval) (イベントの発生周期 )
    [mode](prop_Timer_mode) (モード )

イベント
:   [onTimer](event_Timer_onTimer) (タイマー周期が来た )
