# Math

JavaScript の Math とほぼ互換です。
数学関数を提供します。

# メンバ

メソッド
:   [abs](func_Math_abs) (絶対値の取得 )
    [acos](func_Math_acos) (アーク コサイン )
    [asin](func_Math_asin) (アーク サイン )
    [atan](func_Math_atan) (アーク タンジェント )
    [atan2](func_Math_atan2) (2変数のアーク タンジェント )
    [ceil](func_Math_ceil) (切り上げ )
    [cos](func_Math_cos) (コサイン )
    [exp](func_Math_exp) (eのn乗 )
    [floor](func_Math_floor) (切り捨て )
    [log](func_Math_log) (自然対数 )
    [max](func_Math_max) (二つ以上の値の大きい方 )
    [min](func_Math_min) (二つ以上の値の小さい方 )
    [pow](func_Math_pow) (乗数 )
    [random](func_Math_random) (0以上1未満の乱数 )
    [round](func_Math_round) (整数へのまるめ )
    [sin](func_Math_sin) (サイン )
    [sqrt](func_Math_sqrt) (平方根 )
    [tan](func_Math_tan) (タンジェント )
