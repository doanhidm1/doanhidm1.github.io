# Bitmap.Bitmap

機能/意味
:   大きさを指定したBitmapオブジェクトの構築

タイプ
:   [Bitmapクラス](class_Bitmap)のメソッド

構文
:   Bitmap(width, height, bpp=32)

引数
:   |  |  |
    | --- | --- |
    | width | 画像の幅を指定します。 |
    | height | 画像の高さを指定します。 |
    | bpp | 画像の各ピクセルのビット数を指定します。32 か 8 のいずれかを指定してください。 |

戻り値
:   なし (void)

説明
:   Bitmap クラスのオブジェクトを指定の大きさで構築します。
