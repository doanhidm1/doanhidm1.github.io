# Texture.Texture

機能/意味
:   Bitmapコピー版コンストラクタ

タイプ
:   [Textureクラス](class_Texture)のメソッド

構文
:   Texture(bitmap:Bitmap, format:int=tcfRGBA, is9patch:bool=false)

引数
:   |  |  |
    | --- | --- |
    | bitmap | テクスチャの元となるBitmapクラスのインスタンス |
    | format | カラーフォーマット(tcfRGBA or tcfAlpha), tcfAlpha選択時に色情報はグレイスケール化されます |
    | is9patch | 9patch情報を読み込み、9patch描画用として使用するかどうか |

戻り値
:   なし (void)

説明
