# ImageFunction

ImageFunction クラスは、画像を操作するクラスです。
このクラスからオブジェクトを作成することはできません。

# メンバ

    [colorRect](func_ImageFunction_colorRect) (矩形半透明塗りつぶし )
    [copy9Patch](func_ImageFunction_copy9Patch) (9 path を利用した画像コピー(マルチプラットフォーム版以降) )
    [doBoxBlur](func_ImageFunction_doBoxBlur) (矩形ブラーをかける )
    [doGrayScale](func_ImageFunction_doGrayScale) (グレースケール変換 )
    [drawGlyph](func_ImageFunction_drawGlyph) (文字描画 )
    [drawText](func_ImageFunction_drawText) (文字描画 )
    [fillRect](func_ImageFunction_fillRect) (矩形塗りつぶし )
    [flipLR](func_ImageFunction_flipLR) (左右反転 )
    [flipUD](func_ImageFunction_flipUD) (上下反転 )
    [operateAffine](func_ImageFunction_operateAffine) (アフィン変換演算合成 )
    [operateRect](func_ImageFunction_operateRect) (矩形演算合成 )
