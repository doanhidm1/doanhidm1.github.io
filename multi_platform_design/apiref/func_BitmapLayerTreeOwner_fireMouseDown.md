# BitmapLayerTreeOwner.fireMouseDown

機能/意味
:   マウス押下をレイヤに通知します

タイプ
:   [BitmapLayerTreeOwnerクラス](class_BitmapLayerTreeOwner)のメソッド

構文
:   fireMouseDown(x, y, mb, flags)

引数
:   |  |  |
    | --- | --- |
    | x | マウスのボタンが押された位置の x 座標 ( クライアント座標 ) |
    | y | マウスのボタンが押された位置の y 座標 ( クライアント座標 ) |
    | mb | 押されたマウスボタン |
    | flag | マウスボタンが押されたときに同時に押されていたシフト系のキーの状態 |

戻り値
:   なし (void)

説明
