# Array.load

機能/意味
:   配列をファイルから読み込みます。

タイプ
:   [Arrayクラス](class_Array)のメソッド

構文
:   load(storage:string, mode:string=null) :Array

引数
:   |  |  |
    | --- | --- |
    | storage | 読み込むファイル名 |
    | mode | ファイルを読み込む際のモード文字列を指定します。  "o" に続いてオフセットを10進で指定するとファイルのそのバイト位置からの読み込みになります。 |

戻り値
:   この Array クラスのオブジェクト自身を返します。

説明
:   テキストファイルから配列を読み込みます。
    ファイルのテキストが一行づつ、各要素に入り、最終的に count プロパティはファイルの行数と同じになります。
