# String.split

機能/意味
:   文字列を分割します

タイプ
:   [Stringクラス](class_String)のメソッド

構文
:   split(pattern:string, reserve, ignore=false) :Array

引数
:   |  |  |
    | --- | --- |
    | pattern | 区切り文字か正規表現 |
    | reserve | 予約 |
    | ignore | 空の要素を無視するか |

戻り値
:   文字列を分割した結果が格納された配列オブジェクトを返します。

説明
:   文字列を分割します

参照
:   [Array.split](func_Array_split)
