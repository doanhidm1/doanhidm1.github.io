# Array.remove

機能/意味
:   指定された値と同じ値を持つ要素を削除します。

タイプ
:   [Arrayクラス](class_Array)のメソッド

構文
:   remove(value, all:bool=true)

引数
:   |  |  |
    | --- | --- |
    | value | 削除する値 |
    | all | 全て対象とするか否か |

戻り値
:   なし (void)

説明
:   指定された値と各要素を識別比較し ( === 演算子の動作 )、一致した要素を削除します。
    すべて削除するか、が false の場合は、最初に見つかった要素のみを削除します。
    そうでなければ、見つかったすべての要素を削除します。
