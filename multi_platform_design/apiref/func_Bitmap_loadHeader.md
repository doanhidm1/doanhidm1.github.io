# Bitmap.loadHeader

機能/意味
:   画像情報の読み込み(1.3.0以降)

タイプ
:   [Bitmapクラス](class_Bitmap)のメソッド

構文
:   loadHeader(image)

引数
:   |  |  |
    | --- | --- |
    | image | 読み込む画像ストレージを指定します。  ここで指定するファイル名は拡張子を含んだものになることに注意してください。  拡張子がないとファイルが見つからずエラーとなります。 |

戻り値
:   情報の辞書配列

説明
:   画像情報のみを読み込みます。
    画像の読み込みは行われないため、情報のみを得たい時は高速に読み込めます。
    返される辞書のwidth, heightは必須で、その他の情報は画像形式に依存します。
