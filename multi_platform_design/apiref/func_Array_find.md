# Array.find

機能/意味
:   指定された値が最初に現れる添え字を返します。

タイプ
:   [Arrayクラス](class_Array)のメソッド

構文
:   find(value, offset:int=0) :int

引数
:   |  |  |
    | --- | --- |
    | value | 検索値 |
    | offset | 開始位置 |

戻り値
:   見付かった位置。見付からない時は-1。

説明
:   指定された値を、検索開始位置の添え字から順に見ていき、最初に現れる添え字を返します。
    指定された値が見つからなかったときは -1 を返します。
    比較には識別比較 ( === 演算子の動作 ) が用いられます。
