# BitmapLayerTreeOwner.fireTouchMove

機能/意味
:   タッチが移動されたことをレイヤに通知します

タイプ
:   [BitmapLayerTreeOwnerクラス](class_BitmapLayerTreeOwner)のメソッド

構文
:   fireTouchMove(x, y, cx, cy, id)

引数
:   |  |  |
    | --- | --- |
    | x | タッチ位置の x 座標 ( クライアント座標 ) |
    | y | タッチ位置の y 座標 ( クライアント座標 ) |
    | cx | 指が接触している横方向ピクセル数 |
    | cy | 指が接触している縦方向ピクセル数 |
    | id | タッチID |

戻り値
:   なし (void)
