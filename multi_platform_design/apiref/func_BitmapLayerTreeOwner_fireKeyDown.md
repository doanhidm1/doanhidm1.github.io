# BitmapLayerTreeOwner.fireKeyDown

機能/意味
:   キーが押されたことをレイヤに通知します

タイプ
:   [BitmapLayerTreeOwnerクラス](class_BitmapLayerTreeOwner)のメソッド

構文
:   fireKeyDown(key, shift)

引数
:   |  |  |
    | --- | --- |
    | key | 押されたキーの仮想キーコード |
    | shift | キーが押された時に同時に押されていたシフト系のキーやマウスのボタンの状態 |

戻り値
:   なし (void)

説明
