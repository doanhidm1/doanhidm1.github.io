# Scripts

Scripts クラスは TJS2 に関するメソッドやプロパティを管理します。
このクラスからオブジェクトを作成することはできません。

# メンバ

メソッド
:   [compileStorage](func_Scripts_compileStorage) (ストレージ上のスクリプトのコンパイル )
    [dump](func_Scripts_dump) (コンテキストのダンプ )
    [eval](func_Scripts_eval) (式の評価 )
    [evalStorage](func_Scripts_evalStorage) (ストレージ上の式の評価 )
    [exec](func_Scripts_exec) (スクリプトの実行 )
    [execStorage](func_Scripts_execStorage) (ストレージ上のスクリプトの実行 )
    [getClassNames](func_Scripts_getClassNames) (クラス名を取得 )
    [getTraceString](func_Scripts_getTraceString) (呼び出し履歴の取得 )
    [setCallMissing](func_Scripts_setCallMissing) (missingメソッドの呼び出し設定 )

プロパティ
:   [textEncoding](prop_Scripts_textEncoding) (スクリプトを読み込む時に使用するテキストエンコーディングを指定 )
