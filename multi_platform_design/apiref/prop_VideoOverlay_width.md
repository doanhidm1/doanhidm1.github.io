# VideoOverlay.width

機能/意味
:   [Windows\*]再生矩形の横幅

タイプ
:   [VideoOverlayクラス](class_VideoOverlay)のプロパティ

説明
:   再生矩形の横幅を表します。
    値を設定することもできます。
    このプロパティは、現バージョンではレイヤ再生時は使用できません(常にサイズはビデオのサイズと同じになります)。
    Androidでは常に0を返し、設定しても意味はありません。
