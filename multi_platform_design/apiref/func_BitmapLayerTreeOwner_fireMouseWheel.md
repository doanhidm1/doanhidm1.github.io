# BitmapLayerTreeOwner.fireMouseWheel

機能/意味
:   マウスのホイール回転をレイヤに通知します

タイプ
:   [BitmapLayerTreeOwnerクラス](class_BitmapLayerTreeOwner)のメソッド

構文
:   fireMouseWheel(shift, delta, x, y)

引数
:   |  |  |
    | --- | --- |
    | shift | マウスが移動していた時に同時に押されていたシフト系のキーやマウスのボタンの状態 |
    | delta | ホイールの回転角 |
    | x | ホイールが回転した位置の x 座標 ( クライアント座標 ) |
    | y | ホイールが回転した位置の y 座標 ( クライアント座標 ) |

戻り値
:   なし (void)

説明
