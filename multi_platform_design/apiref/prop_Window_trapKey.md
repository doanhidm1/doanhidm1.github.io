# Window.trapKey

機能/意味
:   [Windows\*]キー入力をトラップするか

タイプ
:   [Windowクラス](class_Window)のプロパティ

説明
:   キー入力をトラップするかどうかを表します。
    値を設定することもできます。
    真に設定すると、他のウィンドウで発生したキー入力イベントを横取りし、このウィンドウ上で発生させることができます。
    複数のウィンドウのこのプロパティが真に設定されている場合は、もっとも最後に作成したウィンドウに対してキー入力イベントが発生します。
    Androidでは常にfalseです。
