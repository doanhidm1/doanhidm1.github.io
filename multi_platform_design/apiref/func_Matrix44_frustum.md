# Matrix44.frustum

機能/意味
:   透視投影変換行列を設定

タイプ
:   [Matrix44クラス](class_Matrix44)のメソッド

構文
:   frustum(left:real, right:real, bottom:real, top:real, znear:real, zfar:real)

引数
:   |  |  |
    | --- | --- |
    | left | 左端 |
    | right | 右端 |
    | bottom | 下端 |
    | top | 上端 |
    | znear | 全面 |
    | zfar | 背面 |

戻り値
:   なし (void)
