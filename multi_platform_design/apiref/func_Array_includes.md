# Array.includes

機能/意味
:   指定された値を含んでいるかどうか判定します。

タイプ
:   [Arrayクラス](class_Array)のメソッド

構文
:   includes(value, offset:int=0) :int

引数
:   |  |  |
    | --- | --- |
    | value | 検索値 |
    | offset | 開始位置 |

戻り値
:   見付かった時は true, 見付からない時は false。

説明
:   指定された値を、検索開始位置の添え字から順に見ていき、指定された値が存在するかどうかを返します。
    比較には識別比較 ( === 演算子の動作 ) が用いられます。
