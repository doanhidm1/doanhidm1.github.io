# Font

Font クラスは、フォントを管理するためのクラスです。

# メンバ

コンストラクタ
:   [Font](func_Font_Font) (コンストラクタ )

メソッド
:   [addFont](func_Font_addFont) (フォントシステムにフォントを追加する。 )
    [getEscHeightX](func_Font_getEscHeightX) (文字の縦方向への X 座標の移動量 )
    [getEscHeightY](func_Font_getEscHeightY) (文字の縦方向への Y 座標の移動量 )
    [getEscWidthX](func_Font_getEscWidthX) (文字の横方向への X 座標の移動量 )
    [getEscWidthY](func_Font_getEscWidthY) (文字の横方向への Y 座標の移動量 )
    [getGlyphDrawRect](func_Font_getGlyphDrawRect) (文字列の実描画範囲 )
    [getList](func_Font_getList) (フォント名の列挙 )
    [getTextHeight](func_Font_getTextHeight) (文字列の縦幅を得る )
    [getTextWidth](func_Font_getTextWidth) (文字列の横幅を得る )
    [mapPrerenderedFont](func_Font_mapPrerenderedFont) (レンダリング済みフォントの割り当て )
    [unmapPrerenderedFont](func_Font_unmapPrerenderedFont) (レンダリング済みフォントの割り当て解除 )

プロパティ
:   [angle](prop_Font_angle) (文字描画角度 )
    [bold](prop_Font_bold) (ボールド )
    [face](prop_Font_face) (フォント名 )
    [height](prop_Font_height) (フォント高さ )
    [italic](prop_Font_italic) (イタリック )
    [rasterizer](prop_Font_rasterizer) (文字列描画方式 )
    [strikeout](prop_Font_strikeout) (取消線 )
    [underline](prop_Font_underline) (アンダーライン )
