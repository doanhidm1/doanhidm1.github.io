# System.screenHeight

機能/意味
:   画面高さ

タイプ
:   [Systemクラス](class_System)のプロパティ

説明
:   画面サイズ ( 画面解像度 ) の縦サイズをピクセル単位で表します。
    値はメインウィンドウのあるディスプレイを対象としたものです。
    メインウィンドウがない場合はプライマリーディスプレイが対象となります。
