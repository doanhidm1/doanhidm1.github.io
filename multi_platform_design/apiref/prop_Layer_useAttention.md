# Layer.useAttention

機能/意味
:   注視情報を使用するかどうか

タイプ
:   [Layerクラス](class_Layer)のプロパティ

説明
:   注視情報を使用するかどうかを表します。
    値を設定することもできます。
    真が指定された場合は、そのレイヤの注視情報が使用されます。
    偽が指定された場合は、そのレイヤの親の注視情報が ( もしあれば ) 使用されます。
