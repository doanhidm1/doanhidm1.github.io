# VideoOverlay.setPos

機能/意味
:   [Windows\*]再生矩形の左上位置を指定

タイプ
:   [VideoOverlayクラス](class_VideoOverlay)のメソッド

構文
:   setPos(left, top)

引数
:   |  |  |
    | --- | --- |
    | left | メディアを再生するための矩形領域の左端位置を指定します。 |
    | top | メディアを再生するための矩形領域の上端位置を指定します。 |

戻り値
:   なし (void)

説明
:   メディアを再生するための矩形領域の左上位置を指定します。
    座標は、ウィンドウのクライアント ( レイヤを表示可能な領域 ) 内での座標で、ピクセル単位で指定します。
