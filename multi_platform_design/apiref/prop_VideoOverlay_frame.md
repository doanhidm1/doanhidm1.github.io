# VideoOverlay.frame

機能/意味
:   [Windows]現在のフレーム

タイプ
:   [VideoOverlayクラス](class_VideoOverlay)のプロパティ

説明
:   メディアの再生位置をフレーム単位で表します。
    値を設定することもできます。
    Androidでは常に0を返し、設定しても意味はありません。
