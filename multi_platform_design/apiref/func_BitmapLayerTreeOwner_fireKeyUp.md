# BitmapLayerTreeOwner.fireKeyUp

機能/意味
:   キーが離されたことをレイヤに通知します

タイプ
:   [BitmapLayerTreeOwnerクラス](class_BitmapLayerTreeOwner)のメソッド

構文
:   fireKeyUp(key, shift)

引数
:   |  |  |
    | --- | --- |
    | key | 離されたキーの仮想キーコード |
    | shift | キーが離された時に同時に押されていたシフト系のキーやマウスのボタンの状態 |

戻り値
:   なし (void)

説明
