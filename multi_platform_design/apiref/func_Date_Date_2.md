# Date.Date

機能/意味
:   年月日時分秒を指定コンストラクタ

タイプ
:   [Dateクラス](class_Date)のメソッド

構文
:   Date(year:int, month:int=0, date:int=1, hour:int=0, min:int=0, sec:int=0)

引数
:   |  |  |
    | --- | --- |
    | year | 年 |
    | month | 月(0～ |
    | date | 日 |
    | hour | 時間 |
    | min | 分 |
    | sec | 秒 |

戻り値
:   なし (void)

説明
:   年月日時分秒を指定して生成
