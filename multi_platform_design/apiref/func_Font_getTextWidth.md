# Font.getTextWidth

機能/意味
:   文字列の横幅を得る

タイプ
:   [Fontクラス](class_Font)のメソッド

構文
:   getTextWidth(text)

引数
:   |  |  |
    | --- | --- |
    | text | 横幅を得たい文字列を指定します。 |

戻り値
:   指定された文字列の横幅がピクセル単位で戻ります

説明
:   このメソッドでは、現在のフォントの設定で指定の文字列を描画したときに必要な横幅を得ることができます。

参照
:   [Font.getTextHeight](func_Font_getTextHeight)
