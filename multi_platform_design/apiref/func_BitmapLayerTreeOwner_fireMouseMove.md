# BitmapLayerTreeOwner.fireMouseMove

機能/意味
:   マウス移動をレイヤに通知します

タイプ
:   [BitmapLayerTreeOwnerクラス](class_BitmapLayerTreeOwner)のメソッド

構文
:   fireMouseMove(x, y, flags)

引数
:   |  |  |
    | --- | --- |
    | x | マウスが移動した位置の x 座標 ( クライアント座標 ) |
    | y | マウスが移動した位置の y 座標 ( クライアント座標 ) |
    | flags | マウスが移動していた時に同時に押されていたシフト系のキーやマウスのボタンの状態 |

戻り値
:   なし (void)

説明
