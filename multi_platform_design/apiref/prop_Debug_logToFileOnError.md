# Debug.logToFileOnError

機能/意味
:   エラー発生時にコンソールのログをファイルに出力するか

タイプ
:   [Debugクラス](class_Debug)のプロパティ

説明
:   真の場合、エラーが発生したときにコンソールのログのファイルへの出力を開始するようになります。
    偽の場合はエラーが発生してもログのファイルへの出力は開始されません。
