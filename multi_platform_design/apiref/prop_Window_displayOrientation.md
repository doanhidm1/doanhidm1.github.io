# Window.displayOrientation

機能/意味
:   ディスプレイの向き

タイプ
:   [Windowクラス](class_Window)のプロパティ

説明
:   ディスプレイの向きを表します。

    * oriUnknown (取得失敗/不明)
    * oriPortrait(縦向き)
    * oriLandscape(横向き)のいずれかの値です。
