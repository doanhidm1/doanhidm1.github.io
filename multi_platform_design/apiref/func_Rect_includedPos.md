# Rect.includedPos

機能/意味
:   矩形内包判定

    right, bottom は含まない(-1された)範囲で判定されます。

タイプ
:   [Rectクラス](class_Rect)のメソッド

構文
:   includedPos(x:real, y:real, matrix:Matrix32) :boolean

引数
:   |  |  |
    | --- | --- |
    | x | X座標値 |
    | y | Y座標値 |
    | matrix | 変換行列 |

戻り値
:   true : 内包する / false : 内包しない

説明
