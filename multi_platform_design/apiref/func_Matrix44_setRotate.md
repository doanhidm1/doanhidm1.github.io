# Matrix44.setRotate

機能/意味
:   回転量設定

タイプ
:   [Matrix44クラス](class_Matrix44)のメソッド

構文
:   setRotate(degree:real, x:real, y:real, z:real)

引数
:   |  |  |
    | --- | --- |
    | degree | 回転角度 |
    | x | 回転軸X |
    | y | 回転軸Y |
    | z | 回転軸Z |

戻り値
:   なし (void)

説明
:   設定のみで現在の行列との演算などは行われません。
    回転軸は正規化推奨
