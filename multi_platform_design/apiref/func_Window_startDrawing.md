# Window.startDrawing

機能/意味
:   明示的に描画を呼び出す。

タイプ
:   [Windowクラス](class_Window)のメソッド

構文
:   startDrawing()

引数

戻り値
:   なし (void)

説明
:   このメソッドはonDrawを間接的に呼び出しますす。
    自分で描画したい場合は、このメソッドを呼び出し、onDraw内で描画します。
    Window.drawCycle を指定して定期的に描画するようにし、このメソッドで明示的に描画することは出来るだけ避けてください。
    将来的に他のプラットフォームでサポートできない可能性があります。
