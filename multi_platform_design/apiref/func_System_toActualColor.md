# System.toActualColor

機能/意味
:   色定数の実際の色の取得

タイプ
:   [Systemクラス](class_System)のメソッド

構文
:   toActualColor(color)

引数
:   |  |  |
    | --- | --- |
    | color | 色定数を指定します ( 色定数一覧 を参照 )。  通常の 0xRRGGBB 形式の色を指定した場合はそのままの値が返ります。 |

戻り値
:   指定された色定数が表す実際の色が 0xRRGGBB 形式で返ります。

説明
:   色定数を実際の色に変換し、0xRRGGBB 形式で返します。
