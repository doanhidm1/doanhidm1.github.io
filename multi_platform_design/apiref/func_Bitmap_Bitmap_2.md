# Bitmap.Bitmap

機能/意味
:   ファイルを指定したBitmapオブジェクトの構築

タイプ
:   [Bitmapクラス](class_Bitmap)のメソッド

構文
:   Bitmap(filename, colorkey=clNone)

引数
:   |  |  |
    | --- | --- |
    | filename | 画像のファイル名を指定します。 |
    | colorkey | カラーキーを指定します。値は Layer.loadImages と同じです。 |

戻り値
:   なし (void)

説明
:   Bitmap クラスのオブジェクトをファイルから構築します。
