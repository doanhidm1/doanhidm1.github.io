# Matrix44.perspectiveFov

機能/意味
:   射影変換行列を設定

タイプ
:   [Matrix44クラス](class_Matrix44)のメソッド

構文
:   perspectiveFov(fovy:real, width:real, height:real, znear:real, zfar:real)

引数
:   |  |  |
    | --- | --- |
    | fovy | 視野角 |
    | width | 幅 |
    | height | 高さ |
    | znear | 全面 |
    | zfar | 背面 |

戻り値
:   なし (void)
