# Window.onCloseQuery

機能/意味
:   [Windows\*]ウィンドウを閉じる確認

タイプ
:   [Windowクラス](class_Window)のイベント

構文
:   onCloseQuery(canclose)

引数
:   |  |  |
    | --- | --- |
    | canclose | ウィンドウを閉じることができるかどうかが渡されます。  下位クラスから上位クラスのイベントハンドラが呼ばれる課程で、下位クラスが決定した「ウィンドウを閉じることができるか」が渡されます。 |

戻り値
:   なし (void)

説明
:   ウィンドウを閉じることができるかどうかを確認するためのイベントです。
    ウィンドウを閉じることができない場合、上位クラスの同メソッドに引数として false を渡してください。
    Androidの場合、タイトル画面での戻るキーで自分で終了する(onKeyXXXを自分でハンドリング)必要があります。
    システムに停止させられる時は、止めようがなく、いきなり終了させられることもあります。
    タスクリストから終了させられるときも問答無用です。
    AndroidではonDeactivateが来た時にオートセーブするしか突然終了に対する対策がありません。
