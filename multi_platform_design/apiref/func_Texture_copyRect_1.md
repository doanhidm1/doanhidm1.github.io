# Texture.copyRect

機能/意味
:   矩形コピー(位置指定)

タイプ
:   [Textureクラス](class_Texture)のメソッド

構文
:   copyRect(dleft:int, dtop:int, src:Bitmap, sleft:int, stop:int, swidth:int, sheight:int)

引数
:   |  |  |
    | --- | --- |
    | dleft | コピー先左端 |
    | dtop | コピー先上端 |
    | src | Bitmapクラスのオブジェクト |
    | sleft | コピー元左端位置 |
    | stop | コピー元上端位置 |
    | swidth | コピー元幅 |
    | sheight | コピー元高さ |

戻り値
:   なし (void)
