# Debug.clearLogFileOnError

機能/意味
:   エラー発生時にコンソールのログファイルをクリアするかどうか

タイプ
:   [Debugクラス](class_Debug)のプロパティ

説明
:   真の場合、エラーが発生したときにコンソールのログを自動的にクリアするようになります。
    偽の場合はクリアはされず、既存のログファイルに追加されます。
