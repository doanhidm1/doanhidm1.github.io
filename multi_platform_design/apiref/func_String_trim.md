# String.trim

機能/意味
:   文字列の先頭と最後の空白を取り除いた文字列を返します

タイプ
:   [Stringクラス](class_String)のメソッド

構文
:   trim() :string

引数

戻り値
:   先頭と末尾の空白文字が削除された文字列を返します。

説明
:   元の文字列には影響を与えません。
    先頭と最後の空白とは、文字コードがU+0001～+0020の文字を指します。
    つまり、ASCII制御文字なども含まれますが、いわゆる全角の空白は含みません。
    連続して先頭または最後にこれらの空白があった場合は、すべて取り除かれます。
