# BitmapLayerTreeOwner.fireDisplayRotate

機能/意味
:   画面が回転されたことをレイヤに通知します

タイプ
:   [BitmapLayerTreeOwnerクラス](class_BitmapLayerTreeOwner)のメソッド

構文
:   fireDisplayRotate(orientation, rotate, bpp, hresolution, vresolution)

引数
:   |  |  |
    | --- | --- |
    | orientation | 画面の向き |
    | rotate | 角度 |
    | bpp | bits per pixel |
    | hresolution | 画面の幅 |
    | vresolution | 画面の高さ |

戻り値
:   なし (void)

説明
