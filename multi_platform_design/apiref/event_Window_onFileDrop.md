# Window.onFileDrop

機能/意味
:   [Windows\*]ファイルがドロップされた

タイプ
:   [Windowクラス](class_Window)のイベント

構文
:   onFileDrop(files)

引数
:   |  |  |
    | --- | --- |
    | files | ドロップされたファイル名が格納された配列(Array)オブジェクトです。 |

戻り値
:   なし (void)

説明
:   ファイルがエクスプローラなどからウィンドウにドロップされたときに発生します。
    単一のファイルがドロップされた場合でも引数には配列オブジェクトが渡されます (最初の要素がそのファイルになります )。
