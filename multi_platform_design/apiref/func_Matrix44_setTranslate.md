# Matrix44.setTranslate

機能/意味
:   移動量設定

タイプ
:   [Matrix44クラス](class_Matrix44)のメソッド

構文
:   setTranslate(x:real, y:real, z:real=0)

引数
:   |  |  |
    | --- | --- |
    | x | 設定するX座標値 |
    | y | 設定するY座標値 |
    | z | 設定するZ座標値 |

戻り値
:   なし (void)

説明
:   行列のXYZ移動位置に値を設定します。
    設定のみで現在の行列との演算などは行われません。
