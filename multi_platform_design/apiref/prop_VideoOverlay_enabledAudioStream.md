# VideoOverlay.enabledAudioStream

機能/意味
:   [Windows\*]再生対象音声ストリーム番号

タイプ
:   [VideoOverlayクラス](class_VideoOverlay)のプロパティ

説明
:   MPEGファイルのみで利用可能です。
    再生対象のオーディオストリーム番号を取得できます。
    オーディオストリームが見付からない場合は-1を返します。
    Androidでは常に0を返す。
