# String.replace

機能/意味
:   文字列の置き換えを行います

タイプ
:   [Stringクラス](class_String)のメソッド

構文
:   replace(pattern:string, replaced:string) :string

引数
:   |  |  |
    | --- | --- |
    | pattern | 正規表現 |
    | replaced | 置き換え後文字列 |

戻り値
:   置き換えられた文字列

説明
:   正規表現パターンに従って文字列の置き換えを行い、置き換えを行ったあとの文字列を返します。
    元の文字列には影響を与えません。
