# Canvas.drawTextureAtlas

機能/意味
:   テクスチャの一部分を描画(シェーダー指定あり)

タイプ
:   [Canvasクラス](class_Canvas)のメソッド

構文
:   drawTextureAtlas(rect:Rect, texture:Texture, shader:ShaderProgram)

引数
:   |  |  |
    | --- | --- |
    | rect | テクスチャ内で描画する矩形領域 |
    | texture | 描画するテクスチャ |
    | shader | 描画に使用するシェーダー |

戻り値
:   なし (void)

説明
