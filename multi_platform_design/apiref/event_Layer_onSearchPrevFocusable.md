# Layer.onSearchPrevFocusable

機能/意味
:   フォーカス可能なレイヤの前方検索

タイプ
:   [Layerクラス](class_Layer)のイベント

構文
:   onSearchPrevFocusable(layer)

引数
:   |  |  |
    | --- | --- |
    | layer | フォーカスを受け取るレイヤオブジェクトです。親クラスの同名のメソッドを呼ぶときに引数として指定します。  null を渡すと、フォーカス可能なレイヤは無いとして処理されます。 |

戻り値
:   なし (void)

説明
:   フォーカス可能なレイヤの前方検索を行うときに発生するイベントです。
    フォーカスを受け取るレイヤを指定するときは、スーパークラスの同名のイベントの引数にそのレイヤを指定してください。
