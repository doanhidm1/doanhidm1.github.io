# Window.onPointerMove

機能/意味
:   何らかのポインティングデバイスが移動された

タイプ
:   [Windowクラス](class_Window)のイベント

構文
:   onPointerMove(type, x, y, cx, cy, shift, id)

引数
:   |  |  |
    | --- | --- |
    | type | タッチもしくはマウス。マウスボタンの種類ではなく、マウスとして通知。 |
    | x | X位置 |
    | y | Y位置 |
    | cx | 接触幅(ピクセル数)、マウスの時は1 |
    | cy | 接触高さ(ピクセル数)、マウスの時は1 |
    | shift | 修飾キー状態(マウス押下状態を含む)、タッチの時は0 |
    | id | タッチID、マウスの時は常に0 |

戻り値
:   なし (void)
