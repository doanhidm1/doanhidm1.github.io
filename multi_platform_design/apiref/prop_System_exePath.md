# System.exePath

機能/意味
:   吉里吉里本体のあるフォルダのパス

タイプ
:   [Systemクラス](class_System)のプロパティ

説明
:   吉里吉里本体が設置してあるパスを表します。
    パス名は統一ストレージ名で表現されます。
    AndroidではpackageCodePathのフォルダ名です。
