# Array.join

機能/意味
:   配列を結合し、一つの文字列にします。

タイプ
:   [Arrayクラス](class_Array)のメソッド

構文
:   join(delimiter:string, reserve, ignore:bool=false) :string

引数
:   |  |  |
    | --- | --- |
    | delimiter | デリミタ。区切り文字。 |
    | reserve | 将来のバージョンのために予約されています。 |
    | ignore | true の場合は、要素中の void は無視されます。 |

戻り値
:   この Array クラスのオブジェクト自身を返します。

説明
:   配列の各要素をデリミタで結合し、結合した文字列を返します。
