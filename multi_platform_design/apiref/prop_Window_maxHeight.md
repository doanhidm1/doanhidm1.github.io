# Window.maxHeight

機能/意味
:   [Windows\*]ウィンドウの最大の縦幅

タイプ
:   [Windowクラス](class_Window)のプロパティ

説明
:   ウィンドウの最大の縦幅を表します。
    値を設定することもできます。
    0を指定すると制限は無くなります。
