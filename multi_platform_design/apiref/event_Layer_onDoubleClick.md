# Layer.onDoubleClick

機能/意味
:   レイヤがダブルクリックされた

タイプ
:   [Layerクラス](class_Layer)のイベント

構文
:   onDoubleClick(x, y)

引数
:   |  |  |
    | --- | --- |
    | x | レイヤがダブルクリックされた位置の x 座標 ( レイヤの表示座標での ) の値です。 |
    | y | レイヤがダブルクリックされた位置の y 座標 ( レイヤの表示座標での ) の値です。 |

戻り値
:   なし (void)

説明
:   レイヤがダブルクリックされた時に発生します。

参照
:   [Layer.onClick](event_Layer_onClick)
