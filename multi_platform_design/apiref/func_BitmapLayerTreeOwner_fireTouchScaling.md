# BitmapLayerTreeOwner.fireTouchScaling

機能/意味
:   拡大操作されたことをレイヤに通知します

タイプ
:   [BitmapLayerTreeOwnerクラス](class_BitmapLayerTreeOwner)のメソッド

構文
:   fireTouchScaling(startdist, curdist, cx, cy, flag)

引数
:   |  |  |
    | --- | --- |
    | startdist | マルチタッチが開始された時のピクセル距離 |
    | curdist | イベント発生時のタッチのピクセル距離 |
    | cx | 中心位置の x 座標 ( クライアント座標 ) |
    | cy | 中心位置の y 座標 ( クライアント座標 ) |
    | flag | マルチタッチ状態フラグです。 |

戻り値
:   なし (void)

説明
