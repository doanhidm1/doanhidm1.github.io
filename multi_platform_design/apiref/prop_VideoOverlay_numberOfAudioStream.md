# VideoOverlay.numberOfAudioStream

機能/意味
:   [Windows\*]音声ストリーム数

タイプ
:   [VideoOverlayクラス](class_VideoOverlay)のプロパティ

説明
:   MPEGファイルのみで利用可能です。
    オーディオストリーム数を取得できます。
    Androidでは常に0を返す。

参照
