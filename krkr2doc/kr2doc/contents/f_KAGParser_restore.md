# KAGParser.restore

機能/意味
:   辞書配列からオブジェクトの状態を復元する

タイプ
:   [KAGParserクラス](f_KAGParser)のメソッド

構文
:   restore(dic)

引数
:   |  |  |
    | --- | --- |
    | dic | 復元するオブジェクトの状態を表わす辞書配列 |

戻り値
:   なし (void)

説明
:   dic で指定した辞書配列を元にして、KAGParser オブジェクトの状態を復元します。

参照
:   [KAGParser.store](f_KAGParser_store)
