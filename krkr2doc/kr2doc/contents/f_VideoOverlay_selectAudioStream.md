# VideoOverlay.selectAudioStream

機能/意味
:   音声ストリームの選択

タイプ
:   [VideoOverlayクラス](f_VideoOverlay)のメソッド

構文
:   selectAudioStream(streamNumber)

引数
:   |  |  |
    | --- | --- |
    | streamNumber | 音声ストリーム番号を指定します。 |

戻り値
:   なし (void)

説明
:   指定した音声ストリーム番号を有効にします。
    音声ストリームを複数含まないビデオでは使用できません。

参照
:   [VideoOverlay.numberOfAudioStream](f_VideoOverlay_numberOfAudioStream)
    [VideoOverlay.enabledAudioStream](f_VideoOverlay_enabledAudioStream)
