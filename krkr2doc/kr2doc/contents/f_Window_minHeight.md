# Window.minHeight

機能/意味
:   ウィンドウの最小の縦幅

タイプ
:   [Windowクラス](f_Window)のプロパティ (読み書き可能)

説明
:   ウィンドウの最小の縦幅を表します。値を設定することもできます。0を指定すると制限は無くなります。
