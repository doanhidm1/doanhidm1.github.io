# VideoOverlay.enabledAudioStream

機能/意味
:   再生対象音声ストリーム番号

タイプ
:   [VideoOverlayクラス](f_VideoOverlay)のプロパティ (読み出し専用)

説明
:   MPEGファイルのみで利用可能です。
    再生対象のオーディオストリーム番号を取得できます。
    オーディオストリームが見付からない場合は-1を返します。

参照
:   [VideoOverlay.selectAudioStream](f_VideoOverlay_selectAudioStream)
    [VideoOverlay.numberOfAudioStream](f_VideoOverlay_numberOfAudioStream)
