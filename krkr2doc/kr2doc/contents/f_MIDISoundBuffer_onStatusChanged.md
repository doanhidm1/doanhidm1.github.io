# MIDISoundBuffer.onStatusChanged

機能/意味
:   ステータスが変更された

タイプ
:   [MIDISoundBufferクラス](f_MIDISoundBuffer)のイベント

構文
:   onStatusChanged(status)

引数
:   |  |  |
    | --- | --- |
    | status | 新しいステータスです。  　[MIDISoundBuffer.status](f_MIDISoundBuffer_status) プロパティを参照してください。 |

説明
:   再生のステータス ( 状態 ) が変わった時に発生します。
