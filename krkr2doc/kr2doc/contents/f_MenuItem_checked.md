# MenuItem.checked

機能/意味
:   チェックマークを表示するかどうか

タイプ
:   [MenuItemクラス](f_MenuItem)のプロパティ (読み書き可能)

説明
:   メニュー項目の横にチェックマークを表示するかどうかを表します。値を設定することもできます。
    　真を指定するとチェックマークが表示されます。
    　[MenuItem.radio](f_MenuItem_radio) プロパティが真の場合は、チェックマークではなく、黒い丸が表示されます。
    　[MenuItem.group](f_MenuItem_group) にて同じグループ番号を持ち、かつ同じ親を持ち、さらに
    [MenuItem.radio](f_MenuItem_radio) プロパティが真のメニュー項目は、
    このプロパティを同時に二つ以上を
    真にすることはできません。他のメニュー項目のこのプロパティが真になった場合、
    他に真になっているメニュー項目があれば、自動的にそのメニュー項目のこのプロパティは偽に
    設定されます。
