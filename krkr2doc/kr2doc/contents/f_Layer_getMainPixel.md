# Layer.getMainPixel

機能/意味
:   メイン画像の色の取得

タイプ
:   [Layerクラス](f_Layer)のメソッド

構文
:   getMainPixel(x, y)

引数
:   |  |  |
    | --- | --- |
    | x | 色を取得する ( レイヤの画像座標での ) x 座標を指定します。 |
    | y | 色を取得する ( レイヤの画像座標での ) y 座標を指定します。 |

戻り値
:   0xRRGGBB 形式の色番号

説明
:   レイヤメイン画像 ( 色を保持している画像 ) の任意の位置の色を取得します。
    　画像座標として無効な ( 範囲外の ) 位置を指定すると例外が発生します。
