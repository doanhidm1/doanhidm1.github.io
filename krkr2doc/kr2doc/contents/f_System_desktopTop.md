# System.desktopTop

機能/意味
:   デスクトップ上端位置

タイプ
:   [Systemクラス](f_System)のプロパティ (読み出し専用)

説明
:   デスクトップ ( ウィンドウを表示可能な領域 ) の上端位置をピクセル単位で表します。
