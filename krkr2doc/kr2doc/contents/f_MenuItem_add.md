# MenuItem.add

機能/意味
:   子メニュー項目の追加

タイプ
:   [MenuItemクラス](f_MenuItem)のメソッド

構文
:   add(item)

引数
:   |  |  |
    | --- | --- |
    | item | 追加する子メニュー項目を指定します。  　子メニュー項目は、すでに子となっているメニュー項目の最後に追加されます。 |

戻り値
:   なし (void)

説明
:   子メニュー項目を追加します。メニュー項目が他のメニュー項目の子になると、
    親メニュー項目が無効化されるときに自動的に子メニュー項目も無効化されるようになります。
    　また、メニュー階層においても親子の関係を持つようになります。
