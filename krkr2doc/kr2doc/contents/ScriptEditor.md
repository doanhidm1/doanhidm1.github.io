# スクリプトエディタについて

スクリプトエディタは、実行中に Shift + F2 を押すことにより表示することができます。
　スクリプトエディタでは簡単な TJS スクリプトの編集とその実行をすることができます。
　このエディタは簡易的なものであり、本格的なスクリプトの編集は意図していません。

　例外が発生したとき、その例外の発生位置を指し示すために「スクリプトエディタ (例外通知)」というスクリプトエディタのウィンドウが開く場合があります。この場合はそのスクリプトエディタは内容が変更禁止の状態になります。

# 画面の説明

![ScriptEditor.png](ScriptEditor.png)

右クリックメニューの説明です。

![CutIcon.png](CutIcon.png) 切り取り
:   選択範囲を切り取ります。切り取られた選択範囲はクリップボードに保持されます。

![CopyIcon.png](CopyIcon.png) コピー
:   選択範囲をクリップボードにコピーします。

![PasteIcon.png](PasteIcon.png) 貼り付け
:   クリップボードから文字を貼り付けます。

![UndoIcon.png](UndoIcon.png) 元に戻す
:   直前の操作を元に戻します。

![ExecIcon.png](ExecIcon.png) 実行
:   スクリプトエディタに入力された内容を TJS2 スクリプトとして実行します。

![SaveIcon.png](SaveIcon.png) 保存
:   内容をファイルに保存します。

![ControllerIcon.png](ControllerIcon.png) コントローラ
:   [コントローラ](Controller) を表示します。

![ExprWatchIcon.png](ExprWatchIcon.png) 監視式
:   [監視式](ExprWatch) を表示します。

![ConsoleIcon.png](ConsoleIcon.png) コンソール
:   [コンソール](Console) を表示します。
