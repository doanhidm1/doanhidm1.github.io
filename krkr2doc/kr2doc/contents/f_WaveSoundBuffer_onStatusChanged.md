# WaveSoundBuffer.onStatusChanged

機能/意味
:   ステータスが変更された

タイプ
:   [WaveSoundBufferクラス](f_WaveSoundBuffer)のイベント

構文
:   onStatusChanged(status)

引数
:   |  |  |
    | --- | --- |
    | status | 新しいステータスです。  　[WaveSoundBuffer.status](f_WaveSoundBuffer_status) プロパティを参照してください。 |

説明
:   再生のステータス ( 状態 ) が変わった時に発生します。
