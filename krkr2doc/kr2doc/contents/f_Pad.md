# Pad

Pad クラスは、簡単なテキストエディタの機能を提供するクラスです。

# メンバ

コンストラクタ
:   [Pad](f_Pad_Pad)

メソッド
:   なし

プロパティ
:   [borderStyle](f_Pad_borderStyle) ( ウィンドウ外見 )
    [color](f_Pad_color) ( 背景色 )
    [fileName](f_Pad_fileName) ( ファイル名 )
    [fontBold](f_Pad_fontBold) ( 太字 )
    [fontColor](f_Pad_fontColor) ( フォントの色 )
    [fontFace](f_Pad_fontFace) ( フォント名 )
    [fontHeight](f_Pad_fontHeight) ( フォントの高さ )
    [fontItalic](f_Pad_fontItalic) ( 斜体 )
    [fontSize](f_Pad_fontSize) ( フォントのサイズ )
    [fontStrikeOut](f_Pad_fontStrikeOut) ( 取消線 )
    [fontUnderline](f_Pad_fontUnderline) ( アンダーライン )
    [height](f_Pad_height) ( ウィンドウの高さ )
    [left](f_Pad_left) ( ウィンドウの左端位置 )
    [readOnly](f_Pad_readOnly) ( 読み込み専用かどうか )
    [showScrollBars](f_Pad_showScrollBars) ( スクロールバーの表示状態 )
    [showStatusBar](f_Pad_showStatusBar) ( ステータスバーを表示するかどうか )
    [statusText](f_Pad_statusText) ( ステータスバーに表示するテキスト )
    [text](f_Pad_text) ( 内容 )
    [title](f_Pad_title) ( タイトル )
    [top](f_Pad_top) ( ウィンドウの上端位置 )
    [visible](f_Pad_visible) ( 表示されているかどうか )
    [width](f_Pad_width) ( ウィンドウの幅 )
    [wordWrap](f_Pad_wordWrap) ( 右端で折り返すかどうか )
