# Layer.onClick

機能/意味
:   レイヤがクリックされた

タイプ
:   [Layerクラス](f_Layer)のイベント

構文
:   onClick(x, y)

引数
:   |  |  |
    | --- | --- |
    | x | レイヤがクリックされた位置の x 座標 ( レイヤの表示座標での ) の値です。 |
    | y | レイヤがクリックされた位置の y 座標 ( レイヤの表示座標での ) の値です。 |

説明
:   レイヤがクリックされた時に発生します。

    [Layer.onDoubleClick](f_Layer_onDoubleClick)
