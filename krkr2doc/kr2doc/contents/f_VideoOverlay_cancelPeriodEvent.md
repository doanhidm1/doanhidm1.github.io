# VideoOverlay.cancelPeriodEvent

機能/意味
:   指定フレームでのイベント発生の解除

タイプ
:   [VideoOverlayクラス](f_VideoOverlay)のメソッド

構文
:   cancelPeriodEvent()

引数
:   なし

戻り値
:   なし (void)

説明
:   setPeriodEventメソッドで設定したイベント発生を解除します。
    この機能は、SWF再生時には利用できません。

参照
:   [VideoOverlay.setPeriodEvent](f_VideoOverlay_setPeriodEvent)
