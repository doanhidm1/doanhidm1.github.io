# Scripts

Scripts クラスは TJS2 に関するメソッドやプロパティを管理します。このクラスからオブジェクトを作成することはできません。

# メンバ

コンストラクタ
:   なし

メソッド
:   [dump](f_Scripts_dump) ( コンテキストのダンプ )
    [eval](f_Scripts_eval) ( 式の評価 )
    [evalStorage](f_Scripts_evalStorage) ( ストレージ上の式の評価 )
    [exec](f_Scripts_exec) ( スクリプトの実行 )
    [execStorage](f_Scripts_execStorage) ( ストレージ上のスクリプトの実行 )
    [getTraceString](f_Scripts_getTraceString) ( 呼び出し履歴の取得 )

プロパティ
:   なし

イベント
:   なし
